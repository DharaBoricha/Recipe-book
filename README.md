🍲 Recipe Book App (Android)
📌 Project Overview

The Recipe Book App is a basic Android application developed using Java and XML. This app allows users to manage and view recipes easily. It is designed as a college project to demonstrate Android development concepts like UI design, data storage, and activity navigation.

🚀 Features

🔐 User Login & Registration (Local Storage)

📋 View list of recipes

➕ Add new recipes

📖 View recipe details (ingredients & steps)

🗑️ Delete recipes

✏️ Edit recipes (optional)

🖼️ Add image for recipes (optional)

📂 Data stored in local JSON file (offline support)

🛠️ Technologies Used

Android Studio

Java

XML (UI Design)

RecyclerView

Local Storage (JSON File)

Gson Library (for JSON parsing)
